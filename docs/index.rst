Quick Start
-----------

.. code-block:: bash

   pip install quantumviz
   quantumviz bloch-sphere input.txt -o output.png

Features
--------

- **Bloch Sphere** - Visualize single-qubit quantum states
- **State City** - 3D density matrix visualizations
- **Cost Landscape** - QAOA/VQE optimization landscapes
- **Circuit Diagrams** - Render quantum circuits
- **Dynamic Flow** - Time evolution animations
- **Dashboard** - Interactive web interface
